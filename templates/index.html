Project Overview

An intelligent machine learning-based system designed to detect nutrient deficiencies in plant leaves through image analysis. This application leverages deep learning algorithms to identify nitrogen, phosphorus, and potassium deficiencies, helping farmers and agricultural professionals make informed decisions about crop health management.

Key Features

Real-Time Camera Capture

Capture leaf images directly using your device camera for instant analysis

Image Upload Support

Upload existing leaf images for comprehensive deficiency analysis

AI-Powered Detection

Advanced MobileNetV2 neural network for accurate deficiency identification

Receive actionable insights and treatment suggestions based on analysis
